# Camera Background Sidebar Menu

This is a basic frontend project built using HTML and CSS.  
It displays a full-screen background image with a hamburger menu (three lines).  
When the menu is clicked, a sidebar opens from the left side.

## Features
- Full screen background image
- Hamburger menu (3 lines)
- Sidebar navigation from left
- Simple and clean design
- Built using only HTML and CSS

## Technologies Used
- HTML
- CSS
